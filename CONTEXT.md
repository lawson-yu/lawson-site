# Lawson Site

个人技术品牌站点，帮助正在学习 AI 与进行工程实践的人理解作者的经验、作品与精选资源。

## Language

**核心受众**:
正在学习 AI、实践软件工程的人；招聘方与技术搜索访客为次要受众。
_Avoid_: 所有人、泛互联网用户

**原创实践内容**:
基于作者真实 AI 或工程经历的复盘、教学与判断；站点内容主线。
_Avoid_: 纯转载、无观点聚合

**精选解读**:
作者筛选 GitHub Trending 项目或 AI 新闻后，补充其解决问题、适用场景与个人判断的内容栏目。
_Avoid_: Trending 搬运、新闻搬运

**作者后台**:
仅站点作者可进入，用于管理文章草稿与发布状态的受保护区域。
_Avoid_: 多作者 CMS、公开投稿后台

**作者身份**:
可进入作者后台的唯一 GitHub OAuth 身份，按稳定 GitHub 用户 ID 授权。
_Avoid_: 任意已登录用户、GitHub 用户名

**草稿**:
仅作者后台可见、尚未对外发布的文章状态。
_Avoid_: 私密文章、未审核文章

**已发布文章**:
对所有访客可见的文章状态；由草稿直接发布。
_Avoid_: 定时文章、审核通过文章

**博客**:
统一承载原创实践、教程与 AI 新闻的文章集合；可按主题标签浏览。
_Avoid_: 纯 AI 新闻页、独立教程系统

**AI 新闻**:
博客中的一类文章，记录与解读 AI 领域动态。
_Avoid_: 独立内容系统、纯转载

**精选项目**:
独立于博客的 GitHub 项目资料库；每项包含仓库链接、解决的问题、用途、技术标签、作者短评与收录日期，可按周聚合。
_Avoid_: 普通博客文章、GitHub Trending 搬运

**自动导入**:
本地 agent 通过受限接口创建草稿的流程；不得直接访问数据库，也不得发布文章。
_Avoid_: 数据库直写、自动发布

**导入密钥**:
仅用于自动导入接口的独立长随机密钥，存于本地 agent 与站点服务端环境变量中，可单独轮换。
_Avoid_: Supabase `service_role` 密钥、GitHub OAuth 凭证

**导入文档**:
一份代表一篇博客、一个精选项目或一个个人项目的 Markdown 文件；YAML front matter 描述 `kind`、`title`、`summary`、`tags` 与来源链接，正文承载内容。
_Avoid_: 批量自由格式文件、直接数据库记录

**内容契约**:
所有导入内容固定拥有 `kind`、`title`、`summary`、`locale`、`tags`、正文与状态；按类型提供项目或精选项目字段，未知字段保留为扩展元数据。
_Avoid_: 无约束自由 JSON、为每种内容复制一套基础字段

**外部内容标识**:
导入文档的强制 `externalId`；同 ID 更新既有草稿而非重复创建。已发布内容收到更新时生成待审核更新草稿。
_Avoid_: 按标题去重、agent 覆盖已发布内容

**部署边界**:
Vercel 承载公开站点、作者后台与受限导入 API；托管 Supabase 承载认证、内容数据与图片。
_Avoid_: 自建常驻服务器、agent 直连数据库

**LAWSON**:
站点公开显示的作者与个人品牌名称，用于首页、作者资料与 `Person` 结构化数据。
_Avoid_: 抽象站点品牌、匿名作者名

**首页定位**:
“LAWSON — AI 与工程实践”；说明站点记录真实项目、拆解 AI 工具与工程问题，并精选值得关注的 GitHub 项目。
_Avoid_: 泛技术博客、无主题的个人主页

**首屏互动视觉**:
首页 Hero 使用原创深色河流背景、透明 AI 原创半身人物与前景浅色河流；桌面端指针经过浅色河流时出现克制波纹，手机端人物居中并保留低频自动波纹。
_Avoid_: 复制第三方人物插画、具体动漫 IP 或工作室风格、依赖未授权人像、人物跟随鼠标或全页持续动效

**首屏视觉语义**:
低饱和电影感的夜色山谷与河流；深蓝黑、雾和少量冷白河光构成安静、有纵深的技术品牌场景，人物侧身望向河流。
_Avoid_: 知识节点、代码符号、彩色数据轨迹、第三方视觉风格复刻

**Markdown 渲染范围**:
支持标题、列表、链接、引用、表格、代码块与图片；禁止原始 HTML 和脚本。代码块提供语法高亮与复制操作。
_Avoid_: 可执行内容、未受控的 HTML

**站内搜索**:
基于 Supabase Postgres 全文搜索已发布内容的标题、摘要、正文与标签，支持关键词和内容类型筛选；搜索结果页不参与搜索引擎索引。
_Avoid_: 客户端全量下载筛选、第三方搜索服务、索引查询页

**标签库**:
内容类型固定，主题标签受后台维护；自动导入匹配已有标签，新标签以待确认状态创建，确认后才公开使用。
_Avoid_: 自由拼写标签、同义标签分裂

**导入包**:
本地 agent 一次提交 Markdown 文件及其相对路径图片；导入服务校验文件后存入 Supabase Storage，并将 Markdown 图片引用改写为受管 URL，再创建草稿。
_Avoid_: 服务器读取 agent 本地路径、agent 直持 Storage 管理密钥、未校验远程下载

**首页**:
公开站点入口，聚合个人定位、最新内容、代表项目与联系方式。
_Avoid_: 单独静态简历页

**项目展示**:
公开呈现作者实践项目的页面，独立于博客与精选项目。
_Avoid_: 精选项目库、文章列表

**SEO 基线**:
每篇公开内容拥有稳定 URL、标题与摘要元数据、Open Graph 图、sitemap 与 RSS；以原创质量和清晰链接结构获取自然收录。
_Avoid_: 关键词堆砌、批量生成低价值页面

**GEO**:
生成式引擎优化，使 ChatGPT、Perplexity 等生成式搜索产品更易理解、检索与引用本站内容。
_Avoid_: 地理位置 SEO、伪造 AI 引用

**GEO 基线**:
公开内容显式回答问题，标注作者与发布日期/更新时间，为外部事实附来源，采用语义化标题，并提供 `Article`、`Person`、`Project` 结构化数据。
_Avoid_: 为 AI 重复生成 FAQ 页面、未验证的引用声明

**默认语言**:
首发公开内容的 `zh-CN` locale；界面不提供语言切换器，文章不要求翻译。
_Avoid_: 无 locale 内容、首发双语维护

**多语言预留**:
所有内容、URL、元数据与导入文档从首发起携带 locale，以支持后续增加英文翻译与语言切换。
_Avoid_: 后期整体迁移、仅在界面层翻译

**视觉语言**:
深色、活泼、内容优先的技术编辑风：近黑底、深海军蓝分区、灰白文本、品牌蓝与洋红栏目强调、亮黄主行动，以及原创河流/山景图形和 AI 原创人物。
_Avoid_: 复制第三方人物、Logo 或插画，通用浅色营销页

**页面动效**:
首屏浅色河流在桌面指针经过时产生低对比度扩散波纹；手机端低频自动波纹；遵从 `prefers-reduced-motion` 并渲染静态首帧。
_Avoid_: 自动播放音效、强制动画、全页持续水流、复制参考站人物动画

**联系入口**:
首页与页脚提供 GitHub、LinkedIn 与邮箱联系链接；首发不提供邮件订阅。
_Avoid_: 空邮件订阅表单、站内消息系统
