# Triage 标签

工程技能使用五个标准 triage 角色；本文件将其映射到本仓库实际使用的标签值。

| 技能中的角色      | 本仓库标签        | 含义                            |
| ----------------- | ----------------- | ------------------------------- |
| `needs-triage`    | `needs-triage`    | 维护者需要评估此任务            |
| `needs-info`      | `needs-info`      | 正等待报告者补充信息            |
| `ready-for-agent` | `ready-for-agent` | 规格完整，可交由自主 agent 执行 |
| `ready-for-human` | `ready-for-human` | 需要人工实现                    |
| `wontfix`         | `wontfix`         | 不会处理                        |

当技能提到某个角色时，使用表中对应的标签值。若将来标签体系变化，只需修改中间列。
