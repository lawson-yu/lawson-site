# 05 — 认证与作者工作区

**What to build:** 唯一作者身份能在统一的 Todo workspace 视觉语言中登录、管理博客/个人项目/精选项目、处理标签和媒体，并预览内容；权限和内容生命周期不变。

**Blocked by:** 01 — 设计基础与公开导航

**Status:** resolved

- [ ] 登录页、作者工作区、内容新建/编辑页、生命周期操作、标签确认、媒体操作和三类预览页采用浅色 hero、双栏工作区与清晰状态呈现；不引入真实 Todo 业务、排序、日期或新持久化。
- [ ] 表单、按钮、链接、输入、状态、上传和删除继续保持现有语义、可见焦点和键盘可用性；不改变作者身份、非作者拒绝、发布/取消发布或数据写入行为。
- [ ] 作者博客、个人项目、精选项目生命周期 e2e 与非作者访问回归通过；真实登录会话下完成 desktop/mobile 工作区巡检。
