# 03 — 公开内容目录与搜索

**What to build:** 访客能在博客、项目、精选项目和搜索页面以高密度 catalogue 视觉浏览、筛选和搜索真实已发布内容，而不改变内容查询或筛选语义。

**Blocked by:** 01 — 设计基础与公开导航

**Status:** resolved

- [ ] 博客、项目与精选项目列表，以及搜索页，按 Catalogue 的 grid、toolbar、metadata、卡片与响应式层级重构；不添加课程、远程筛选、假分页或假数据。
- [ ] 有真实封面时按响应式媒体比例展示；无封面时保持清晰文本卡片，不生成占位图或复用不相关的 Hero 资源。
- [ ] 现有搜索、内容类型筛选、链接、空结果和键盘可访问行为通过浏览器回归；desktop/mobile 均无非预期横向滚动。
