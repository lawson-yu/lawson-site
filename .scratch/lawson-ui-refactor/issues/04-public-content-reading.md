# 04 — 公开内容详情阅读

**What to build:** 读者能以低干扰的 Article Detail 阅读布局浏览博客、个人项目和精选项目详情，同时继续使用现有 Markdown、代码、元数据、图片和真实外部链接。

**Blocked by:** 01 — 设计基础与公开导航

**Status:** resolved

- [ ] 三类公开详情使用统一的阅读层级、metadata、正文、深色相关内容区和 responsive detail layout；不截断正文，也不将内容改造成课程文章或增加虚构相关推荐。
- [ ] Markdown 标题、列表、链接、引用、表格、代码块、复制操作和受管图片保持可用；项目的技术栈、问题、产出和 repository/demo 链接继续可达。
- [ ] 公开详情 e2e 与浏览器巡检确认 desktop 长阅读、小屏堆叠、键盘焦点和无横向滚动均保持可用。
