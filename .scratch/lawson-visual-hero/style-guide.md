# 深色河流互动首屏：人物美术规范

## 风格名称

柔软 3D 漫画玩偶

## 一句话定义

以圆润简化造型、哑光软胶/黏土质感和柔和低对比光影呈现的原创 3D 漫画人物；气质温暖、安静、亲切，适合深色河流场景。

## 风格特征

- 画面类型：3D 风格化角色渲染，不是 2D 半厚涂或照片级写实。
- 造型语言：大头小身、圆润轮廓、低到中等几何复杂度；五官极简、友好、具有轻微立体阴影。
- 光影：柔和正面或侧前方主光，低对比、阴影边缘柔软；避免硬边赛璐璐阴影和强烈轮廓光。
- 色彩：低至中饱和度，暖灰、奶油白、深棕与少量暖色；需要与深蓝黑河流背景形成清晰但克制的分离。
- 材质：皮肤、头发和衣物均为干净哑光表面；带细微体积、漫反射和柔软触感，避免塑料高光、金属感或粗糙写实纹理。
- 情绪：安静、友好、略带思考感；不卖萌过度，不使用夸张大笑、哭泣或惊讶表情。
- 构图：透明背景半身素材，人物侧身望向右下；腰部以下可被浅色河流前景遮挡。

## 可复用关键词

3D 风格化角色、柔软漫画玩偶、圆润比例、简化五官、哑光软胶、黏土质感、柔和体积光、低对比光影、暖灰服装、深棕蓬松短发、干净边缘、安静亲切、低饱和电影感、透明背景。

## 生成约束

- 只生成原创人物；不复刻或点名任何具体 IP、角色、工作室或艺术家。
- 仅吸收通用的 3D 柔软玩偶视觉语言，不复制参考图的发型、服装、姿势、表情、背景或彩色放射线。
- 输出透明背景单人半身素材；不含河流、山、建筑、道具、底座、文字、Logo、边框或水印。
- 角色身体轻微侧向右下并望向右下；轮廓完整，四肢和手部自然，不出现粘连、重复或畸形。
- 使用哑光软胶/黏土质感与柔和低对比光；避免平面 2D 插画、真人照片、强反光塑料和高饱和霓虹。
- 人物需适配深色河流背景：浅色或暖灰上衣、深棕头发、清晰透明边缘。

## 生图风格前缀

原创柔软 3D 漫画玩偶风格角色，圆润简化比例，友好克制的极简五官，干净哑光软胶与细腻黏土质感，柔和侧前光，低对比阴影与自然体积感，低至中饱和暖灰配色，安静、亲切、有思考感；透明背景单人半身素材，清晰干净边缘。
