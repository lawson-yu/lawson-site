# 02 — 加入桌面河流波纹交互

**What to build:** 桌面访客将鼠标移到人物前的浅色河流时，可获得克制的冷白扩散波纹反馈；人物、背景和首页内容不随鼠标移动。

**Blocked by:** 01 — 上线静态河流首屏

**Status:** resolved

- [x] 波纹仅在浅色河流交互区域内产生，指针移出区域后不新增波纹。
- [x] 每次交互最多两圈低对比波纹，约 0.8 秒自然消散，不覆盖人物或扩展至整张背景。
- [x] 动画按真实帧时间推进，资源清理后不保留循环或定时任务。
- [x] 公开行为测试验证交互活动状态与消散后的静止状态，不绑定 Canvas 内部绘制细节。

## 答案

- `RiverHero` 内部 Canvas 使用真实帧时间推进，限制两枚活动波纹，并在空闲后取消动画帧。
- 验证：`pnpm typecheck`；`pnpm exec playwright test e2e/public-experience.spec.ts --grep '桌面端浅色河流'`。
